
# 🥗 SlimMom — Healthy Lifestyle & Calorie Tracker

SlimMom este o aplicație React modernă dedicată utilizatorilor care doresc să își monitorizeze alimentația, să își calculeze necesarul caloric zilnic și să își gestioneze produsele consumate. Proiectul demonstrează abilități solide în React, routing, state management, formulare complexe și UI responsiv.

🔗 **Demo Live:** https://anisoara1.github.io/SlimMom/

---

## ✨ Funcționalități principale

- **Calculator caloric personalizat** pe baza datelor introduse de utilizator.
- **Gestionarea produselor consumate**: adăugare, listare și ștergere.
- **Autentificare și sesiune persistentă** (dacă proiectul include partea de auth).
- **Pagini multiple** cu navigație fluidă.
- **Design modern și responsiv**, optimizat pentru mobil și desktop.
- **Validare formulare** și feedback vizual pentru utilizator.
- **Persistență locală** pentru datele introduse (în funcție de implementare).

## 🔍 Detalii despre implementare

### Calculator caloric
Utilizatorul introduce:
- vârstă  
- greutate  
- înălțime  
- obiectiv (slăbire / menținere)  

Aplicația calculează necesarul caloric zilnic folosind formule standardizate.

### Gestionarea produselor
- listă de produse consumate în ziua curentă  
- adăugare produs cu nume + calorii  
- ștergere produs  
- total calorii consumate  

### Navigație
Aplicația include pagini precum:
- Home  
- Calculator  
- Diary / Products  
- Login / Register (dacă sunt implementate)
